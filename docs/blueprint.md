# **App Name**: HealthInsight Registry

## Core Features:

- 重要结果录入: 提供一个全面的表单，用于记录患者基本信息、体检详情、具体异常结果（A/B类、详细信息）以及初步处置/通知信息。支持日期/时间的默认填写和手动修改，并自动将当前登录用户指定为通知人。
- 个人信息补充管理: 一个专门的界面，用于填写或更新患者的姓名、性别、年龄和电话号码等个人详细信息。该界面可在初次结果录入后自动进入，也可通过单独的患者管理功能直接访问。
- 体检报告PDF上传与关联: 允许用户上传患者的PDF体检报告，指定检查日期，分类文件类型，并安全地将这些报告与相应的异常结果记录关联起来，便于检索和查阅。
- 数据浏览、搜索与编辑: 提供一个集中的用户界面，供用户查看所有已记录的异常结果和患者数据，具备强大的搜索和过滤功能，并能够按需修改现有记录。
- 基础数据统计与展示: 实现根据输入的异常结果数据生成和显示基本统计报告或摘要的功能，帮助用户进行总体概览和分析。
- 核心数据库交互: 建立与集中式MySQL 8.4数据库安全且稳定的通信连接，管理所有数据的读取、写入、更新和删除操作，确保数据的一致性和可靠性。

## Style Guidelines:

- Primary color: 专业而沉稳的蓝色 (#2962FF)，象征数据严谨性与可靠。背景色: 与主色调同源的浅淡蓝色 (#EDF2F8)，提供柔和、清晰的视觉基础。点缀色: 充满活力的紫罗兰色 (#834BFF)，用于突出重要操作和关键信息，同时保持专业氛围。
- 使用 'Inter' 字体作为主要字体，适用于所有标题和正文。'Inter' 是一种无衬线字体，具有现代、清晰、客观的外观，非常适合处理大量数据和医疗信息，确保了极佳的可读性和专业感。
- 采用清晰、临床风格的图标，以帮助用户快速理解医疗数据类别和操作，优先考虑用户界面的可用性和直观性。
- 设计干净、结构化的表单和数据表格布局，强调信息的明确划分、高效的数据输入流程，以及患者记录的易读性。
- 采用微妙且功能性的动画效果，为数据保存、表单验证和记录导航等用户操作提供视觉反馈，确保流畅的用户体验而不会造成干扰。
